Название: test_avito.py
Версия	: v1.4

Файлы в архиве:
readme.txt 			- инструкция по использования скрипта
test_avito.py 		- сам скрипт
Result.csv 			- мои результаты прохождения с дефолтными значениями
ход_выполнения.doc 	- история создания скрипта
Extra				- папка с дополнительными материалами, там же версии скрипта по мере готовности, если интересно

test_avito.py создавался для автоматической записи заголовка, ссылки на товар, стоимости, имени, адреса и телефона продавца из объявления с сайта avito.ru по заданным критериям в файл .csv.

Скрипт был написан и протестирован на python v3.9.0 и chromedriver v87.0.4280.20. Работоспособность на других версиях не гарантируется.
Для корректной работы скрипта вам понадобится установленные на компьютер python 3 и chromedriver.
Также для корректной работы скрипта вам потребуется установить в настройках chrome devtools мобильное устройство под названием Nexus 5,
подробнее о том как это сделать: https://docs.google.com/document/d/10R-S7cElFbrMijynopBr3XArm99GwaCb-Rg6cQ7UHHI/edit

Инструкция пользования данным скриптом:
1. Откройте скрипт в любом редакторе кода (например: notepad++, sublime text и тд.);
2. На 21 строке кода заменить путь к файлу chromedriver на свой:
	browser = webdriver.Chrome('ващ_путь_к_файлу_chromedriver.exe');
3. На 73 строке кода заменить путь для сохранения файла Result.csv
	f = open('/Users/nikolay/Documents/Selenium/Autotests/Result.csv', 'w')
4. В скрипте по умолчанию заданы 4 переменные:
	Категория товара: catName = '  Коллекционирование'
	Важно: 
		Если категория обобщенная, например "Недвижимость", то пишем без 2 пробелов - ('Недвижимость')
		Если конкретная подкатегория, например "Квартиры", то пишем с 2 пробелами в начале - ('  Квартиры')
		Вводить данные между кавычек
	Название города: cityName = 'Новосибирск'
	Важно: 
		Название города должно быть написано правильно с большой буквы
		Вводить данные между кавычек
	Поисковый запрос: input_text = 'lego'
	Важно:
		Вводить данные между кавычек
	Минимальная стоимость товара: min_price = 10000
	Важно: 
		Можно использовать только числовые значения
		Вводить данные без кавычек
5. Достаточно запустить скрипт test_avito.py в командной строке и дождаться завершения;
6. Файл с данными Result.csv будет создан в директории Selenium или в той директории, которую вы до этого использовали.
